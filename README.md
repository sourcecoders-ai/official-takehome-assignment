# Team Member Title Bug Exercise

## Problem Description
When editing a team member's title, the changes are not being saved. The title field remains unchanged after updating.

## Steps to Reproduce
1. Create a new team member with any title
2. Edit the team member and change their title
3. Save the changes
4. Observe that the title remains unchanged

## Expected Behavior
The title should update to the new value when editing a team member.

## Task
Investigate why the title field is not being updated and implement a fix.

## Testing Your Solution
1. Create a test team member
2. Edit their title
3. Verify the change persists

## Submitting Your Solution
1. Create a branch: `git checkout -b fix/team-member-title`
2. Make your changes
3. Test thoroughly
4. Create a PR with:
   - Description of the bug
   - Your solution approach
   - How you tested it

## Success Criteria
- Title updates work correctly
- Existing functionality remains intact
- Code is clean and documented
- Tests pass

Good luck!
